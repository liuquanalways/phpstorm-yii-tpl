# phpstorm-yii-tpl
A phpstrom code template for Yii2

----------------------------
## Usage

put the Yii.php file below the .PhpStorm2017.1\config\fileTemplates\ directory.

## example

file->new->Yii
fill File name and TYPE(m, v, c)

* m --- Model template
* v --- View template
* c --- Controller template
